# air md
Markdown with Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'air_md'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install air_md

## try example
```bash
bundle exec ruby lib/air_md.rb example/basic.air.md
```

```bash
bundle exec ruby lib/air_md.rb input_file [output_file]
```
